# First Principles — documentation (français, bref)

**Version : Beta 1.0**

**First Principles** est un projet **Unity 6** qui associe une **calculatrice graphique** (fonctions, dérivée numérique sur une grille) et un **plateformeur 2D** : plateformes et dangers suivent la courbe et sa dérivée, avec une **progression par étapes** et des thèmes (séries, polar, intégrales / sommes de Riemann, ingénierie, etc.).

## Documentation complète en anglais

Le guide détaillé (installation, gameplay, mathématiques, concours) est surtout en **anglais** pendant l’ajout progressif d’autres langues.
